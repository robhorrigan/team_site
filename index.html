GrooveWith.Me

A Rails application where users create and share social playlists.

  • Implemented Spotify API to authenticate users and create playlists.
  • Enabled Twilio gem texting to share secret user code with others to join an event and share their playlist.

    • Github Live Site

4DaysOut.com

Rails application for short-term planning of dinner and movie dates.

  • Interacted with Google Movies via web scraping with Nokogiri and Open URI.
  • Utilized Yelp API to geo-locate restaurants near desired locations.

    • Github Live Site
